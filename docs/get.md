## get

### Get Element by id
```javascript
const foo = dom.getId('foo')
```

### Get Multiple Elements by id
```javascript
const [ foo, bar, baz ] = dom.getId('foo', 'bar', 'baz')
```

### Get Elements by Class Name
```javascript
const bar = dom.getClass('bar')
```

### Get Elements by Tag Name
```javascript
const divs = dom.getTags('div')
```
